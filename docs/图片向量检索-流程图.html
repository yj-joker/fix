YIELD node AS f, score
WHERE score >= $minScore
OPTIONAL MATCH (c:Component)-[:CAUSES]->(f)
OPTIONAL MATCH (f)-[:HAS_SOLUTION]->(s:Solution)

💡 跨模态原理

故障照片
→
multimodal-embedding-v1
→
1024维向量 A
"轴承磨损"
→
multimodal-embedding-v1
→
1024维向量 B

向量 A 和向量 B 在同一向量空间中，可以计算余弦相似度
cosine(A, B) 越大 → 图片内容与文字描述越相关

五、两套向量空间（互不干扰）

embedding（纯文字检索）

  • 模型: text-embedding-v4
  • 用途: 用户输入文字 → 检索相似故障/部件
  • 字段: embedding (Component, Fault)
  • 索引: fault_embedding_index, component_embedding_index
  • 调用方: Java EmbeddingUtils 直接调阿里云
  • 场景: "轴承过热" → 搜索文字描述相似的故障

imageEmbedding（跨模态检索）

  • 模型: multimodal-embedding-v1
  • 用途: 图片↔文字 跨模态检索
  • 字段: imageEmbedding (全部5种实体)
  • 索引: *_image_index (5个)
  • 调用方: Java → Python /ai/embedding/image
  • 场景: 拍照 → 搜索视觉相似的故障图片

⚠️ 为什么不能混用？

text-embedding-v4 生成的向量只能与 text-embedding-v4 的向量比较（纯文字空间）。
multimodal-embedding-v1 生成的向量（无论输入是图片还是文字）在同一个跨模态空间中。
两个模型的向量维度可能相同，但语义空间完全不同，跨模型比较无意义。

七、核心方法说明

Python 端

方法 / 接口位置说明
POST /ai/embedding/image api/main.py 接收图片URL列表，调 multimodal-embedding-v1 返回每张图的1024维向量
POST /ai/embedding/text-multimodal api/main.py 接收文字，用同一模型转为跨模态向量（与图片在同一空间）
search_by_image_vector() services/graph_service.py 通用方法：在指定实体类型的 image_index 中做向量检索
query_diagnosis_by_image_vector() services/graph_service.py 在 fault_image_index 检索 → OPTIONAL MATCH 扩展到 Component + Solution
GraphImageSearchTool._execute() tools/graph_query_tool.py Agent RAG 工具：图片→向量→平均→图谱检索→格式化返回

Java 端

方法位置说明
getImageEmbedding(List<String>) ImageEmbeddingUtils 调 Python /ai/embedding/image，拿到多图向量后逐维求平均，返回 List<Double>
save() / update() 5个 ServiceImpl 若 imageUrls 非空，调 ImageEmbeddingUtils 生成 imageEmbedding 后存入 Neo4j
getFaultsByImageEmbedding() FaultRepository @Query 注解，调 db.index.vector.queryNodes('fault_image_index', ...)
getComponentByImageEmbedding() ComponentRepository @Query 注解，调 db.index.vector.queryNodes('component_image_index', ...)
findDiagnosisPathsByImage() GraphQueryServiceImpl 核心方法：图片→向量→fault_image_index检索→图谱路径扩展→DiagnosisPathVO
searchByImage() PathController POST /weixiu/path/image-search 入口，转发给 GraphQueryService
